## 二、免训练方案 (Ready-to-Use)

### A. 推荐首选

#### 1. inswapper_128_fp16 (Deep-Live-Cam)

| 指标 | 评分/数据 |
|------|----------|
| **效果** | ★★★★ (4/5) |
| **FPS (4060)** | 25-35 |
| **显存占用** | ~2GB |
| **训练需求** | 无 |
| **实时可行** | ✅ 是 |
| **开源** | ✅ 是 (MIT) |

**Pipeline 分解 (1080p 单脸, RTX 4060):**

| 阶段 | 模型 | 耗时 |
|------|------|------|
| 人脸检测 | SCRFD-10GF | ~4ms |
| 人脸对齐 | 2D-106 | ~1ms |
| 特征提取 | ArcFace | ~5ms |
| 换脸融合 | inswapper_128_fp16 | ~15ms |
| 后处理 | Gaussian Blur | ~2ms |
| 合成输出 | - | ~5ms |
| **总计** | - | **~32ms (31 FPS)** |

**优点:**
- FP16 精度，模型仅 ~380MB
- RTX 4060 实测 25-35 FPS
- 开箱即用，无需训练
- 边缘平滑，表情保留好
- 活跃开源社区

**缺点:**
- 换脸区域固定 128x128
- 极端角度效果下降
- 光照剧烈变化时不稳定

**推荐度:** 🥇 **首选**
