# FaceVideoChange

## What This Is

一款面向主播/UP主的实时视频流换脸工具。输入摄像头视频流 + 目标人脸图片，输出换脸后的视频流，支持 RTMP 推流和本地录制。

## Core Value

主播能在直播/创作中实时将自己的脸换成目标人脸，延迟低于 100ms，效果和速度均衡。

## Requirements

### Active

- [ ] 实时换脸视频通话 — 视频会议/直播中实时把自己的脸换成目标人脸
- [ ] 摄像头推流输出 — 连接摄像头，实时处理后通过 RTMP 推流到直播平台
- [ ] 本地文件录制 — 换脸后的视频流保存为视频文件
- [ ] 目标人脸特征提取 — 上传人脸图片后提取特征，一次提取多次使用
- [ ] 人脸相似度检测 — 多张图片上传时自动检测是否为同一人
- [ ] 人脸质量评估 — 检测图片是否适合换脸（脸部遮挡、角度、清晰度）

### Out of Scope

- 图形界面（GUI）— 优先实现功能，CLI 为主
- 移动端支持
- 视频会议 SDK 深度集成（Zoom/腾讯会议）
- WebRTC 方案（先以 RTMP 为主）

## Context

- **硬件环境**：Windows PC，RTX 4060（8GB 显存），32GB 内存
- **技术偏好**：优先使用成熟开源方案，如无合适方案可自行训练（推理需在消费级 GPU 上运行）
- **用户画像**：主播/UP主，用于创作内容和娱乐直播

## Constraints

- **延迟**：端到端延迟 < 100ms
- **推理硬件**：RTX 4060（8GB VRAM）能流畅运行
- **质量均衡**：既要有可接受的换脸效果，也要保证实时性
- **开源优先**：优先评估现有开源换脸方案
